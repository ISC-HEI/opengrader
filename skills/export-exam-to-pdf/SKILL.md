# Export Exam YAML to PDF

This skill exports an exam YAML file into PDFs ready for GradeScope upload. It generates both a blank exam template and student answer PDFs with normalized page counts.

## When to Use This Skill

Trigger this skill when the user mentions:
- "export to PDF" / "generate PDFs" / "create PDF"
- "GradeScope" combined with "PDF" or "exam"
- "convert exam to PDF"
- "exam template PDF"
- "student answer PDFs"

## Workflow

### Step 1: Locate Exam YAML File

If the user did not give you the location of the YAML file, ask for it.


### Step 2: Generate PDFS with the python script

Run the pdf generation script with the correct parameters. 
The parameters are the following :
- `-i` : The path to the exam YAML file. You should already have this at this step
- `-o` : The path to the output folder to put all the generated markdown. You can put this in a subfolder of the YAML exam file named `pdf/`

```bash
uv run generate_pdfs.py -i <path-to-exam.yaml> -o <output-dir>
```

This scripts will take a long time to finish, it is normal. Do not timeout on this step

At the end of this step, you should have something like :

`./<yaml_exam_folder>/<filname>.yaml`
`./<yaml_exam_folder>/pdf/**` -> multiples pdf files generated by the tool


### Step 6: Organize Output

Final output structure:
```
<output-dir>/
├── <exam>.yaml
├── pdf/
│   ├── Firstname_Lastname.pdf
│   └── ...
```

## Bundled Scripts

### scripts/generate_template.py
Export PDFS from the content of a YAML exam file. Export one PDF for each student, and a template PDF.

## Example Usage

User: "Can you export my exam to PDFs for GradeScope?"

1. Ask for the path of the exam YAML
> User answer (e.g., `./opengrader/exam.yaml`)
2. Run: `uv run generate_pdfs.py ./cc_1/exam.yaml ./cc_1/pdf`
3. Run: `uv run generate_markdown.py ./opengrader/exam.yaml ./output/markdown`
4. Confirm with user that PDFs are ready for GradeScope upload
